# Extract scalar metadata from a modelblueprint

Extract scalar metadata from a modelblueprint

## Usage

``` r
extract_target(x, ...)

extract_yhat_name(x, ...)

extract_exposure_name(x, ...)

extract_exposure_value(x, ...)

extract_exposure_zero_rep(x, ...)

extract_offset_name(x, ...)

extract_offset_value(x, ...)

extract_display_name(x, ...)

extract_deploy_notes(x, ...)
```

## Arguments

- x:

  A `modelblueprint`.

- ...:

  Unused.

## Value

The slot value (`NA` when not set).
