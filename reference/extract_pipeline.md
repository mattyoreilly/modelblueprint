# Extract pipeline functions from a modelblueprint

Extract pipeline functions from a modelblueprint

## Usage

``` r
extract_pre_process_fun(x, ...)

extract_feat_eng_fun(x, ...)

extract_post_process_fun(x, ...)
```

## Arguments

- x:

  A `modelblueprint`.

- ...:

  Unused.

## Value

The pipeline function.
