# Replace feature name vectors in a modelblueprint

Replace feature name vectors in a modelblueprint

## Usage

``` r
set_original_inputs(x, value, ...)

set_feature_names(x, value, ...)
```

## Arguments

- x:

  A `modelblueprint`.

- value:

  A character vector of column names.

- ...:

  Unused.

## Value

A new `modelblueprint`.
