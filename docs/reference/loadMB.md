# Load a modelblueprint from disk

Reconstructs a `modelblueprint` from a `.tar.gz` archive created by
[`saveMB()`](saveMB.md).

## Usage

``` r
loadMB(path)
```

## Arguments

- path:

  Path to the `.tar.gz` archive created by [`saveMB()`](saveMB.md).

## Value

A fully reconstructed `modelblueprint` object.
